## 七、安全机制

### 7.1 密码安全 🔐

加密算法：使用 bcrypt 算法对密码进行哈希加密

随机盐值：每个密码使用随机盐值，防止彩虹表攻击

不可逆性：密码哈希不可逆，即使数据库泄露也无法获取明文

### 7.2 API 限流 🛡️

限流框架：使用 Flask-Limiter 限制请求频率

请求限制：默认每分钟 60 次请求

防护目的：防止恶意请求和 DDoS 攻击

### 7.3 匿名保护 🎭

昵称隐藏：匿名广场帖子只显示随机昵称

设备绑定：匿名昵称由 device_id hash 生成

一致性：同一设备每次发帖显示相同匿名名

隐私保护：其他人无法追踪到真实用户身份

### 7.4 数据传输安全 🔒

传输加密：使用 HTTPS 加密传输（推荐）

客户端处理：敏感数据（如密码）在客户端预处理

输入校验：服务端对所有输入进行校验

## 八、核心功能技术实现

### 8.1 心情记录与可视化

心情趋势图：使用 Canvas 绘制心情趋势折线图，支持 emoji 标记

心情映射：7 种心情对应不同颜色和 emoji

心率同步：支持蓝牙心率设备数据同步

云端同步：数据自动同步到云端服务器

### 8.2 心理测评系统

量表实现：PHQ-9、GAD-7、PSS、睡眠质量 4 份量表

自动评估：根据得分自动计算等级和给出建议

结果可视化：使用 BarChartView 柱状图对比各量表得分

趋势分析：使用 TrendChartView 折线图展示历史趋势

云端同步：测评结果自动同步到服务器

### 8.3 匿名广场系统

帖子列表：RecyclerView + SwipeRefreshLayout 实现列表

图片上传：支持图文混排，图片压缩后上传

排序功能：服务端 SQL 排序（时间/热度）

点赞系统：点赞/取消点赞使用设备ID去重

回复系统：回复支持多层嵌套

### 8.4 番茄钟系统

计时器：CountDownTimer 实现倒计时功能

状态切换：自动切换专注/休息模式

数据持久化：SharedPreferences 按日期归档数据

统计功能：自动计算今日/本周统计数据

宠物联动：每个番茄为宠物增加 25 点经验

### 8.5 桌宠系统

动画绘制：自定义 PetView 绘制宠物动画

数据模型：Pet 数据模型存储等级、经验值

经验获取：心情记录、打卡、学习等行为获取经验

升级系统：经验值满自动升级，解锁新外观

情绪联动：宠物根据用户心情做出不同反应
